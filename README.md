# Aniversariantes Feitosa

> A Vue.js project

## Setup

- Install dependencies

```sh
yarn install
```

- Serve with hot reload at localhost:8080

```sh
yarn run start
```

- Build for production with minification

```sh
yarn run build
```

- Build for production with minification

```sh
yarn run build
```

- Deploy

```sh
yarn run deploy
```
